# Lights Out

### Setup and usage
Clone this repo 
``` git clone git@github.com:jbevis/lights-out.git```

Cd into the new local repo:
``` cd lights-out```

Install dependencies
``` yarn install```
or
``` npm install```

Run the app locally
```yarn start```
or

```npm run start```

Run Tests
```yarn test```
or
```npm run test```

#### Objective

Try to turn off all the illuminated squares to win.

Go for a personal best in the fewest amount of moves.
